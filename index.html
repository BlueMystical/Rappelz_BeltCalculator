    Features:

    • Can Select Any Pet or Soul Pet.
    • Can Select Any Boss Card.
    • Some belt slots lets you choose only pets, some only boss cards, and some lets you select anything.
    • There is an option to switch between selecting a Pet or a Boss card (if allowed).
    • Red Numbers are the % in excess a certain stat has, so you can use that excess in some other stat.
    • Green Numbers show flat values granted by Boss cards or indirect stat bonus gained from other stats,
      for example: 1 Str = 2,8 P.Atk and 10 Max.Weight. (flat values are not Percentages).
    • Yushiva Belt option removes the 1% bonus extra gained for having more than 1 of the same pet stat equiped.
    • Death Gladiator can be belted.
    • All cards are illustrated.
    • Responsive design, fully tested in Chrome, Firefox and Android Mobile devices.
